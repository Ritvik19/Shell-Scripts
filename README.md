# Shell-Scripts
Shell Scripts for data processing

* `head` : prints first n rows
    * `-n` [number of rows]
    * `-f` [file path]
* `tail` : prints last n rows
    * `-n` [number of rows]
    * `-f` [file path]
* `columns` : prints header of all columns
    * `-f` [file path]
* `unique` : prints unique values of a column
    * `-n` [column number]
    * `-f` [file path]
* `value-counts` : prints value counts of a column
    * `-n` [column number]
    * `-f` [file path]
